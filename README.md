# Summer 2027 Tech Internships

A list of tech internships for Summer 2027 and off-season terms (fall, winter, spring) in the US. Maintained by me for now.

> ⭐ **Star this repo** to save it and get updates when new roles are added.

## scope

- software engineering, data and ML, hardware, quant, and product roles
- US-based positions (US-based remote is fine)
- Summer 2027 and off-season terms

Roles outside this are out of scope. See [CONTRIBUTING.md](CONTRIBUTING.md) before adding one.

## about

I'm US-based, so for now this list is US only and built for me. If it ever gets more stars, I'm open to changing it based on what people using it need, including adding Canada. I want to keep it active and update it daily. If at some point I can't, I'd rather hand it to someone who can keep it going than let it go inactive. Probably won't come to that. For now I made it for myself and for anyone else who needs it.

## legend

- 🔒 closed
- 🛂 does not offer sponsorship
- 🇺🇸 requires US citizenship
