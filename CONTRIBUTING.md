# contributing

Thanks for adding a role. Keep it simple.

## what belongs here

- tech internships: software engineering, data and ML, hardware, quant, product
- US-based (US-based remote is fine)
- Summer 2027 or off-season (fall, winter, spring)
- a link to apply

## what to skip

- roles outside tech
- roles outside the US
- duplicates already in the list
- closed roles: mark them 🔒 instead of deleting the row

US only is just for now. I'm US-based and built this for myself. If the list grows, I'm open to adding Canada and other regions based on what people using it need.

## how to add one

Two ways.

1. Open an issue with the company, role, location, and link. I will add it.
2. Or edit `README.md` and open a pull request. Add a row to the table in this format:

   ```
   | Company | Role | Location | [apply](https://link) | 2026-06-08 |
   ```

   Put new rows at the top. Date Posted is the date the role went up, in YYYY-MM-DD.

## fixes and corrections

Found a broken or wrong link, a missing role, or an attribution that's off or missing? Open an issue or pull request. I'll review and make the changes.

## markers

Add these to a row when they apply:

- 🔒 closed
- 🛂 does not offer sponsorship
- 🇺🇸 requires US citizenship

That is the whole guide.
